pwd - 查看当前工作目录 - print working dierectory

/ - 根目录 - 相当于windows下的计算机

/root - 超级管理员的主目录

/home/aaa - 普通用户的主目录/home

ls -查看文件和文件夹 - list directory contents

以.开头的文件夹是隐藏文件夹

ls -a：显示所有文件 -all

ls -l：长格式查看

mkdir aaa：创建文件夹aaa

  • -p - 在需要父文件的时候一次性创建
  • rmdir -删除空文件夹 - remove emoty directory

cd - 切换目录 - change directory

  • 绝对路径 - 以根路径/作为访问的基路径

  • 相对路径 - 以当前路径作为访问基路径

  • ~当前用户的用户主目录，如果是root用户就是/root

    touch - 如果文件不存在就创建文件，如果存在就更新创建时间

rm - 删除文件 - remove

  • -rf:删除文件夹，不执行询问

  • r：删除文件夹（递归式的删除） -reccursive

  • -f ：强制删除（不提示） -force

  • -i ：交互式删除（询问） - interactive

cp - 拷贝文件 - copy
-r ：递归的复制 ，复制文件夹
mv - 移动文件

rename ：批量改文件后缀（rename .htm.html*.htm）将htm改为html

cat - 连结多个文件的内容并显示 - concatenate

cat -n ：可以显示行号

iconv -f gb2312 -t utf-8 文件1 > 文件2（将文件转换新的文件并传入文件2，文件2存在就修改内容，不存在就新建文件2）

head -100 文件名:查看文件开头100行

tail -10 文件名:查看文件后10行

head -10 文件名 | cat -n：显示文件前10行并显示行号

cat -n 文件名 | tail -10：显示最后十行并显示行号

| ：管道可以将前后内容连起来

more（less） 文件名 ：显示一屏内容，分屏查看

rev 文件名：每行内容都反过来看

tac 文件名：从下往上看

grep 字符串：所搜文件内容，有特殊符号，字符串加引号

-E “<script\s+.*>“:正则表达式

find / -name “”*.html” ：所有根目录的.html的文件

  • -size +1k：大于1kb的文件
  • ctime -7d：创建时间7天内的文件
  • atime:文件最后访问时间
  • mtime：文件修改时间
  • exec 执行额外的操作

wc:统计行数
